ts-dynamic-query 目的是为了结构化描述筛选条件，主要用于 typescript 筛选内存中数组数据，当然如果不是特别需要这种表述行的条件筛选可以使用 lodash。

## 目标

1. 可以使得筛选条件结构化
2. 可序列化/反序列保存我们的筛选条件

## 安装

```bash
$ npm install ts-dynamic-query --save
```

```bash
$ git clone https://github.com/wz2cool/ts-dynamic-query-demo.git
```
